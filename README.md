# Link Pick mod
Helps you pick ores more convenient.

Minecraft版本: 1.7.10 

Mod latest version: 0.1.1.200323_beta

基于FML和gradle构此项目

后期目标:
1. 对1.8.9以及更高Minecraft java版提供支持
2. 添加config可自定义搜索范围
3. 提供对第三方mods支持
4. etc.
